# ai-jobs-market-dashboard
SQL + Power BI analysis of 1,500 AI job postings across 14 countries — uncovering salary trends, hiring patterns, and market insights for 2025–2026.

## Key Findings
- AI Solutions Architects lead at $251K avg salary
- Hybrid roles dominate at 46% of all AI positions
- Big Tech pays $237K vs Startups at $168K
- USA accounts for 515 out of 1,500 roles

## Tools Used
- SQL (SQLiteOnline)
- Power BI Desktop
- Kaggle Dataset: AI Jobs Market 2025–2026

## Files
- /sql — 7 SQL query results
- /dashboard — Power BI .pbix file
